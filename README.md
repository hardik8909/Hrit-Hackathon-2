# 🌍 Vayunetra — Smart Air Quality & Health Advisory System

## 🚀 Overview
Vayunetra is an intelligent system that monitors air quality in real-time and provides personalized health advisories based on environmental conditions. It also includes an admin dashboard for monitoring data, managing alerts, and controlling the system.

---

## 🎯 Problem Statement
Air pollution is a major health risk, but people often lack real-time awareness and actionable insights to protect themselves.

---

## 💡 Solution
Vayunetra provides:
- 📊 Real-time Air Quality Index (AQI) monitoring  
- 🧠 AI-based health recommendations  
- 📍 Location-based pollution tracking  
- ⚠️ Alerts and safety suggestions  
- 🛠️ Admin dashboard for system monitoring

- 👥 User/activity monitoring  
- 📈 View analytics and reports
  
## 🧠 AI Logic
The system uses AQI thresholds to generate health advice:

- AQI < 50 → Good air quality
- AQI 50–100 → Moderate
- AQI 100–200 → Wear mask
- AQI > 200 → Stay indoors
